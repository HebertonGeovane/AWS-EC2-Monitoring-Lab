Monitoramento e Logs no Amazon EC2

Laboratório prático utilizando Amazon EC2, CloudWatch, Metric Filters e SNS

Arquitetura de Monitoramento AWS

Passo a passo da atividade

  1. Atualizar a instância EC2
  2. Instalar o CloudWatch Agent
  3. Configurar coleta de logs
  4. Criar Log Group no CloudWatch
  5. Criar Metric Filter para detectar ERROR
  6. Criar Alarme HighErrorCount
  7. Criar tópico SNS
  8. Gerar logs de erro
  9. Criar alarme de CPU HighCPUUsage
  10. Gerar stress de CPU

Objetivo do laboratório

Implementar um sistema completo de monitoramento de logs e métricas utilizando serviços AWS.

  • Logs enviados para CloudWatch
  • Metric Filter detecta ERROR
  • Alarmes configurados
  • Notificação via SNS
  • Monitoramento de CPU
